Publications

  • Yizhu Jiao, Yun Xiong, Jiawei Zhang and Yangyong Zhu. Collective Link Prediction Oriented Network Embedding with Hierarchical Graph Attention. CIKM. 2019 [paper] [slide]

    Abstract: Aligned social networks provide more information to alleviate data insufficiency. In this paper, we target on the collective link prediction problem to predict both the intra-network social links and the inter-network anchor links. The major challenges involve the problems of network characteristic differences and link directivity differences. To address this problem, we propose an application oriented network embedding framework, Hierarchical Graph Attention based Network Embedding. It learns the node representations by aggregating information from both the intra-network neighbors and inter-network partners. What’s more, we introduce a hierarchical graph attention mechanism to resolve the two challenges mentioned before.
